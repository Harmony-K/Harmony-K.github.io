Overview

A website specifically designed for people who want to give D&D a shot. To be clear, it doesn't aim to teach people how to play D&D, but will aid with the creation of simple characters which ultimately make learning the game much easier. The best way to learn is by playing.

Furthermore, It also aims to aid with gameplay - it’s easier to start off playing D&D with a more basic character so that a newbie can get used to the mechanics. Thus, specific classes and races will be available.

This website is being designed as a tool for beginners. Players who find that they are comfortable with the gameplay are encouraged to use another app or site that is more advanced (there are SO MANY out there and they have way more cool stuff). The site will also provide FAQs and other various tools/popups implemented throughout the site.

Once again, there are a lot of sites where you can create a character in no time, and for players who know the system and how things work, this is super useful and efficient to use. However, for beginners it’s a different story. Ultimately, this site will streamline and simplify the character creation process and make the concepts of the game easier to grasp.

Motivation

If you’ve ever played D&D, you know that creating a character from scratch takes time, and if I’m being honest, it discourages a lot of new players. Look, the character creation process can be a lot of fun. However, if it's your first time playing, there is a lot to take in. Some people get overwhelmed by the amount of content there is available and it can be quite intimidating. This is where a lot of people lose interest and give up.

Ok so let’s say you've made a character, but you don't know how to play. And on top of that, the character sheet is hard to read. AND you have too many things to keep track of. These are prime examples of why people never end up playing the D&D. Hopefully, this website will help remedy these problems and we can get more people playing the game and having fun with friends (and, hopefully, making friends too!).

Description

A website seems like the best option for now. Rather than an app, the website will be available to more people (not restricted to a certain OS) as it can be accessed through various devices.

The website will consist of four pages; character creation, character sheet desktop / game mode, a mobile phone character sheet / game mode and a ? page (for FAQs). It will be recommended that the character creation is to be done on a desktop though. Maybe in the future, this project can develop into a responsive website.

The reasoning behind the separate pages for editing and “game mode” is that there is nothing worse than trying to perform a task and then accidentally deleting or editing something important. Making a webpage specifically for a mobile phone just makes accessing and viewing “character sheets” easier. It will have all the same content as the desktop view, but will have buttons with headings separating content and, once clicked, will show the relevant content.

As this website aims to help beginners, only levels 1-5 for the main classes in 5e Dungeons and Dragons will be available. While it might seem like it will exclude a lot of ”cool” options the game has to offer, ultimately, it will benefit the user when first learning how to play.

Character Creation Page

  • Ability scores and hit points:
  • Bio / Personality traits / Appearance

What is needed?

From the research that I’ve done and from conversations with a friend, as far as I know, the resources needed include:

  • A computer - With access to the internet. To run the software needed, write the code and run the website.
  • Github - To host the website and be able to work effectively in a team.
  • Visual Studio Code - Makes life easier when writing code and accessing/organising files.
  • Node.js - to read and run React.js and styled-components, so the website can be interactive.

The skills required for this project include; the ability to write in HTML, CSS, JavaScript and use GitHub correctly. All the software listed above are open source. As classes are online, it would be safe to assume everyone has access to a computer and the internet, and that we have just learnt HTML and CSS. As such, the project seems very feasible.

Outcome

Hopefully, we get more people playing D&D, having fun and making memories. The character creation will become super easy and efficient, and the simplified character sheets will help beginners continue playing. The learning curve is lowered, but there is still some customisation that allows for a more in-depth character to be built.

To sum it up, the website simplifies the character creation phase and makes accessing the content you need from the character sheet easier. From there, once players are comfortable and used to gameplay, they can move onto other applications and websites with more advanced features and content.
